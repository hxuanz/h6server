========================================================================
    控制台应用程序：h6server 项目概述
========================================================================
服务端程序 -- 基于boost asio
--------------------------------------------------------
usage: h6ocr -i port -o log_dir

port    指定启动端口， 默认12345
log_dir 指定日志输出目录
--------------------------------------------------------
---- 支持的接口
ocr/xhd
自动识别血化单。

/asr
语音识别 [目前仅支持 采样频率16000，单声道的 wav格式音频]
--------------------------------------------------------
---- 开发指南
依赖：
1. boost   -- asio网络库， 其他方便的处理函数
2. jsoncpp  -- json处理
3. opencv2  -- 图像切割处理
4. tesseract-build  -- ocr引擎
6. msc（科大讯飞）  -- asr引擎（需联网）


server文件夹： 基于asio搭建的网络框架
util文件夹： 自己写的方便工具
api文件夹： 接口逻辑实现。目前只有 ocr和asr

master.cpp: 任务分发，根据请求参数的不同，调用不同的处理逻辑（api）
