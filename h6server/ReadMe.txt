========================================================================
                           h6server 项目概述
========================================================================
服务端程序 -- 基于boost asio
--------------------------------------------------------
usage: h6ocr -i port -o log_dir

-i 指定启动端口， 默认12345
-o 指定日志输出目录【必须】
--------------------------------------------------------


支持的接口：
1. ocr/xhd   -- 自动识别血化单。
2. /asr  --语音识别 [目前仅支持 采样频率16000，单声道的 wav格式音频]
--------------------------------------------------------


依赖的库：
1. boost   -- asio网络库， 其他方便的处理函数
2. jsoncpp  -- json处理
3. opencv2  -- 图像切割处理
4. tesseract-build  -- ocr引擎
6. msc（科大讯飞）  -- asr引擎（需联网）
--------------------------------------------------------


server文件夹： 基于asio搭建的网络框架（参考的官方示例，无需大改动）
util文件夹： 自己写的工具函数
api文件夹： 接口逻辑实现。目前只有 ocr和asr， 【可扩展】

master.cpp: 任务分发，根据请求参数的不同，调用不同的处理逻辑（api） 【可扩展】
